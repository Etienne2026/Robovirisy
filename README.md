
**Robovirisy 9.7.11.25**  
================================

**I- INTRODUCTION**  
Robovirisy est un mini-programme conçu pour supprimer les virus informatiques. Il élimine les virus présents sur l’ordinateur et sur les clés USB infectées, et il est capable de restaurer les données cachées par un virus.  
Il fonctionne en mode silencieux quand il est activé en version Premium. Après 30 jours d’activation, il passe automatiquement en mode Premium.

**II- SYSTÈMES COMPATIBLES**  
Les systèmes compatibles avec Robovirisy sont : Windows 7, 8, 10 et 11.  
Sur les versions antérieures, Robovirisy peut provoquer des erreurs inconnues et ne fonctionne pas correctement.

**III- ACTIVATION**  
Vous bénéficiez de 5 jours d’essai gratuit pour protéger votre ordinateur et réparer les clés USB connectées. Après 5 jours, vous devrez l’activer pour débloquer toutes les fonctions.

1. **Comment activer Robovirisy ?**  
Après 5 jours, une fenêtre s’affiche en haut de l’horloge. Il y a une case de saisie de code : saisissez-le et appuyez sur "Activer", puis attendez l’affichage d’un message.  
Si le code est correct, l’activation se fait automatiquement. Sinon, l’activation est refusée en cas de faute de frappe ou de code non autorisé.

2. **Comment obtenir un code valide ?**  
Sur cette fenêtre, un numéro Mobile Money s’affiche. Envoyez 20 000 Ar à ce numéro avec comme référence l’ID affiché sur la fenêtre. Vous recevrez immédiatement un code par SMS dès réception du paiement.

3. **Après 5 jours**  
Certains virus sont ignorés par Robovirisy en version d’essai. L’activation est nécessaire pour détecter tous les virus.

**IV- CONDITIONS D’UTILISATION**  
Ce programme demande une autorisation en tant qu’administrateur. Robovirisy doit modifier certains paramètres de votre système d’exploitation et se lance automatiquement au démarrage de l’ordinateur.  
En l’acceptant ou en l’exécutant, vous en assumez l’entière responsabilité.

**NB :**
- Avant d’utiliser Robovirisy, assurez-vous que votre système est compatible.
- Désactivez temporairement l’antivirus pendant l’installation si besoin : Robovirisy contient des scripts sensibles aux antivirus. Si un script est bloqué, cela peut causer des dysfonctionnements imprévus.
- Pour les utilisateurs de Windows 7 : vérifiez que votre système n’est pas déjà infecté par le virus Wininit.exe dans les processus utilisateur. S’il n’est pas présent, vous pouvez utiliser Robovirisy. Si vous n’êtes pas à l’aise avec l’informatique, gardez Windows Defender actif.

**V- FONCTIONNEMENT**  
Robovirisy s’exécute en arrière-plan. Quand il détecte un virus, il affiche une notification visuelle ou sonore et le supprime.  

**Pour les virus de type raccourci - Astuce supplémentaire :**  
1. Installez Robovirisy  
2. Insérez la clé USB infectée et attendez que vos données soient récupérées.
